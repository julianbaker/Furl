- Menu bar items that overflow off-screen stay reachable. The Furl icon lists them; pick one and it slides on-screen next to the icon until you're done with it.
- Furl never opens or activates items. You click them yourself.
- Excluded items stay permanently visible to the right of the icon. Furl re-establishes positions automatically as apps come and go.
- Accessibility is the only permission. No Screen Recording, no background networking, no telemetry, no auto-updater.

Polish in this release:

- Resolved visual bugs related to revealed icon position.
- Prevent menu bar items from disappearing while being hovered.
- Menu populates reliably on launch.
- Granting Accessibility no longer requires a relaunch.
- No more silent errors. If an item can't be revealed, Furl lets you know.
